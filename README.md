# BEGINNER JAVA PROJECT

## 🎵 Music Theory Quiz (CLI App)

A multiple-choice quiz application written in Java that tests knowledge of musical terminology such as tempo and
dynamics.

This project was built to demonstrate core Java fundamentals including control flow, collections, file handling, and
user input validation.

---

## 🎬 Demo

[Watch demo on YouTube](https://www.youtube.com/watch?v=cvQxc9d3LB8)

---

## ✨ Features

- 📂 Loads questions from a CSV file
- 🎲 Randomised question order
- 🧠 Multiple-choice answers (A–D)
- ✅ Input validation (prevents invalid answers)
- 📊 Score tracking at the end of the quiz
- 🏷️ Category selection (Tempo or Dynamics)
- ⚠️ Handles invalid data in CSV gracefully

---

## 🛠️ Technologies Used

- Java
- IntelliJ IDEA
- CSV file handling (`BufferedReader`, `InputStream`)
- Enums for type safety (`Option`, `Category`)

## What I learned

- How to structure a Java application beyond a single main method
- Using enums to represent fixed sets of values
- Reading and parsing data from external files (CSV)
- Handling user input and validation in a CLI environment
- Separating logic from input/output for cleaner code
- Basic error handling to make the app more robust
